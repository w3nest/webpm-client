# @w3nest/webpm-client

Library for dynamic npm's libraries installation from W3 Nest ecosystem.

* <a href='https://w3nest.org/apps/@webpm-client/doc/0.1.14-wip' target='_blank'>Documentation</a>
* <a href='https://w3nest.org' target='_blank'>W3Nest</a>

## Installation, Build, Test

To install the required dependencies:

```shell
yarn
```

---

To build for development:

```shell
yarn build:dev
```

To build for production:

```shell
yarn build:prod
```

---

To run tests:

```shell
yarn test
```

To measure test coverage:

```shell
yarn test-coverage
```

---

To generate code's documentation:

```shell
yarn doc
```
